## Summary

Formé Haus is a luxury fashion e-commerce storefront built on Shopify Hydrogen. It features a dark, premium aesthetic with sophisticated animations and full Arabic/English bilingual support.

## Key Findings

### Architecture
- **Framework:** Remix 2.17 + Hydrogen 2024.10 + React 18
- **Styling:** Tailwind CSS 4 + Framer Motion animations
- **Deployment:** Shopify Oxygen (edge workers)
- **i18n:** English + Arabic with RTL support

### Project Structure
- 157+ files in `app/` directory
- 50+ React components
- 30+ route files (localized)
- Custom UI context for global state
